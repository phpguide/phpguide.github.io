מדריך עדכון גרסת php בשרת wamp

wamp - חבילה הכוללת את כל הרכיבים הנדרשים לפיתוח. אמנם היא נוחה מאוד, אך תמיד עם גרסאות מיושנות יותר של התוכנות.


כל גרסת php באה לתקן באגים, להוסיף אפשרויות חדשות ולשפר אלגוריתמים ישנים. אין ספק שחשוב להישאר מעודכנים בגרסה האחרונה של שפת התכנות ואם זה תלוי לא רק בכם, אלה ב-wamp למשל — הגיע הזמן לסדר את העניינים.

העלאת גרסה ב-wamp



1. הורדת php


גשו לאתר ההורדות של php והורידו את הגרסה האחרונה ביותר. שימו לב להוריד את גרסת ה- VC9. גרסת ה-vc6 מיועדת לשרתים ישנים יותר.
בחרו בארכיון zip של Thread Safe (הבלוק השני) והורידו אותו.


2. פתיחת הארכיון


כשתפתחו את הארכיון תקבלו את התיקיה php-5.x.x-Win32-VC9-x86
שנו את שמה ל- php5.x.x כשם הגרסה. למשל php5.3.6

3. הכנסת הגרסה לתוך wamp


נעתיק את התיקיה הזו שלנו לתוך C:/wamp/bin/php/. שם יכולות להיות תיקיות עם גרסאות אחרות של php (5.3.3) למשל.

4. הגדרות לגרסה החדשה


כנסו לתיקיה של הגרסה הישנה, זאתי שהייתה קיימת. 5.3.3 למשל.
מצאו בא את הקבצים php.ini , phpForApache.ini , wampserver.conf
והעתיקו אותם לתיקיה עם הגרסה החדשה.

פתחו את שלושת הקבצים האלה (בתיקיה החדשה) עם עורך טקסט כלשהו
notepad רגיל יתאים לחלטוין. מצאו בכל אחד מהקבצים בנפרד את מספר הגרסה הישנה
והחליפו אותו במספר הגרסה החדשה.

אם הגרסה הישנה הייתה 5.3.3 והחדשה 5.3.6,
מצאו בשלושת הקבצים את כל המקומות שבהם מופיע המספר 5.3.3 והחליפו אותו במספר 5.3.6. הכי נוח לעשות את זה עם קיצור המקלדת ctrl + H


5. הפעלה השרת מחדש


סגרו את wamp לגמרי והפעילו מחדש. עכשיו התפריט יכלול אפשרות לבחירת גירסה חדשה של php. בחרו את הגרסה החדשה יותר ושוב הפעילו את שרת ה-apache מחדש.

6. בדיקת השינויים


ניצור ונריץ סקריפט עם הפקודה phpinfo() ונבדוק שהגרסה הכן התעדכנה. זהו זה, עכשיו הכל בסדר ואפשר לעבוד עם הגרסה החדשה בכיף :)
