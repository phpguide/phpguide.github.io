Yii עברו ל-GIT והוציאו גרסה חדשה - 1.1.10

משתמש_112077,
 16 לפברואר 2012
יצאה גרסה חדשה (1.1.10) של Yii, ובנוסף, הקוד של הפרימוורק עבר ל-GitHub.


המעבר אמנם קרה בערך שבוע אחרי יציאת הגרסה, אבל אף אחד לא סיפר על זה עדיין.

הגירסה החדשה (1.1.10) כוללת תיקוני באגים, שבעצם זו מטרת כל תתי הגירסאות של 1.1 (לתקן באגים).

הפעם מדובר בבאגים עם CActiveForm, CJuiDatePicker ובעיה עם הפעלת yiilite.
רשימת השינויים המלאה נמצאת כאן.

בנוסף, כל הקוד של הפריימוורק עבר מ-google code (איפה הוא התאחסן עד עכשיו ב-SVN) אל GitHub וניתן לצפיה, הורדה ופורק (Fork) בכתובת https://github.com/yiisoft/yii.

אפשר לציין שאצל אותו משתמש GitHub נוצר רפוזיטורי חדש בשם Yii 2, ואולי הקוד של Yii 2 יהיה זמין לצפיה עוד לפני יציאתה של גרסה 2 שעד עכשיו אף אחד לא ידע מה בעצם הולך איתה. את גרסה 2 הבטיחו לנו בראשון לינואר, אבל במקום זה קיבלנו את 1.1.9.

עכשיו גירסה 2 מובטחת לנו לקראת סוף מרץ. נראה אם זה באמת יקרה או שזו רק בדיחה לקראת הראשון באפריל.
