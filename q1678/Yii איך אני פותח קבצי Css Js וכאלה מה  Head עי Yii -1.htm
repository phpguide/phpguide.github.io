מצטער, אבל כנראה שבזמן הקרוב אתם תשמעו ממני הרבה לגבי Yii, כי התחלתי ללמוד על הפריימוורק הזה.

איך אני פותח קבצי Js ו Css ע"י Yii, לא הבנתי את הקוד שיש ב Github של PHPguide כאן:
<?php Yii::app()->clientScript->registerCssFile($this->getAssetsBase().'/styles/style.css'); ?>


כי הפונקציה getAssetsBase לא קיימת ב Yii, אז איפה שמים את הפונקציה, ואיך מגדירים לה שכל הקבצים יהיו בתקייה Static כמו ב PHPguide?, ועוד משהו, למה ב PHPgoide ב HomepageController אין Layout? אם אין, אז איך יכול להיות שהוא מציג את החלק העליון והשאר?
