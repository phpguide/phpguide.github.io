מה יעיל יותר:
function person(name){
  this.name=name;
  this.speak=function(str){
    //some code
  }
}

או
function person(name){
  this.name=name;
}
person.prototype.speak=function(str){
  //some code
}

הבנתי שהשיטה השניה תופסת פחות זכרון, משום שהפונקציה לא נוצרת מחדש בכל יצירה של instance
האם זה נכון?
האם יש יתרונות לשיטה הראשונה?
