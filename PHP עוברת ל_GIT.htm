מערכת שמירת היסטוריית קוד מקור ופיתוח בצוות- git שסיפרתי עליה כל כך הרבה בזמן האחרון מחליפה את מערכת ה-svn עבור קוד המקור של מפענח PHP


כנראה ש-git הופכת לסטנדרת של מערכות פיתוח בצוות ובדחיפה חזקה מוציאה את ה-SVN אל מחוץ לתמונה . יותר ויותר פרויקטים פופולארים עושים את הבחירה ב-GIT וגם תורה של PHP הגיעה.

לאחר שבועיים של דיונים והצבעות הגיעו המפתחים לתוצאה הבאה:
52 מהמפתחים הצביעו בעד המעבר ל-GIT
15 אל Mercurial
1 אל Bazaar
ו-13 העדיפו להישאר עם svn

מאז 2008 יש העתק של הקוד ב-github אבל אף אחד לא מתכוון באמת להשתמש בו.
חברי צוות המפתחים הרימו שרת git משלהם ויבצעו את כל העבודה בכתובת git.php.net/

להזכירם דיברנו על GIT כאן
וגם סיפרנו שהקוד מקור של האתר הזה נמצא ב-git
