1) בשביל להעלאות תמונה (או כל קובץ אחר) תשתמש באינפוט מסוג file (לפעמים תאלץ לתת לטופס את התכונה enctype="multipart/form-data")
2) בשביל להעלאות את התמונה לשרת (כקובץ או כמידע) תצטרך להשתמש במערך הגולבאלי $_FILES וכדאי לעלות את הקובץ לשרת עם move_uploaded_file.
http://www.w3schools.com/php/php_file_upload.asp
3) אם אתה רוצה לאחסן את התמונה במסד מבלי להעלות אותה לשרת, תשתמש בקידוד base64.
http://www.namepros.com/webmaster-tutorials/81935-upload-image-to-mysql-database-php.html
