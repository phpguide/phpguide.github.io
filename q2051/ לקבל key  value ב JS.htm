לקבל key + value ב JS

פתח soogo , 31 לאוקטובר 2014
איך אפשר לקבל key + value של מערך כזה
Properties = { 'Text': 'string', 'Size': 'int', 'Color': 'string' };

1 תשובות

ענה Splash ב 31 לאוקטובר 2014 #

צריך ליצור לולאה שתרוץ על כל המפתחות של המערך,ואז כדי לגשת לערך הולכים למערך במפתח X.
משהו בסגנון הזה -

Properties = { 'Text': 'string', 'Size': 'int', 'Color': 'string' };
for(key in Properties){
alert(key+' : ' + Properties[key]);
}
