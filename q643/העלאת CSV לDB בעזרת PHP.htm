ב PHP יש פונקציה מובנית לקריאת קבצי csv שהעבודה איתה מתבצעת באותה צורה כמו קריאת קובץ רגיל:

$handle = fopen("test.csv", "r");
while($data = fgetcsv($handle))
{
 print_r($data);
}


data זה המערך שלך עם כל הנתונים מאותה שורה.
את ההכנסה למסד אתה עושה בצורה הרגילה שאתה עושה עם הנתונים מתוך המערך.
