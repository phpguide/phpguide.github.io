NaseemIB96,

תמיד רציתם לקצר זמן טעינת הדף שלכם? אתר חדש עם ממשק פשוט יעזור לכם - הוא יקצר את זמן הטעינה על ידי הקטנת קובץ ה-CSS שלכם.


כל מה שקראתם עד עכשיו תוכלו לעשות במספר צעדים:
1 - תכנסו לאתר הבא: http://www.csscompressor.com/.
2 - ב-Compression Mode תשנו להכי גבוה (highest).
3 - ל-CSS Input תעתיקו את הקוד של קובץ ה-CSS שלכם.
4 - ללחוץ על Compress, ובום - הקובץ נהיה קטן יותר, בכמה? כתוב כשלוחצים על Compress.

טיפ: תמיד יש לשמור קובץ גיבוי, מכיוון שאחרי שמקטינים את הקובץ הוא נהפך לקשה לקריאה.

ניסיון קטן, תוצאות נהדרות:
לקחתי את קובץ ה-CSS של האתר הזה והקטנתי, והופתעתי לגלות את התוצאות הבאות:
Compression ratio: 62.219KB/97.007KB = 35.9% (-34788 Bytes)

מה שאנחנו רואים כאן:
המשקל לפני שהקובץ הוקטן: 97.007KB
המשקל אחרי שהקובץ הוקטן: 62.219KB
הפרש: 34788 Bytes
הפרש באחוזים: 35.9%
