לא, זה לא יגרום לדף להיתקע
שנית, תהפוך את כל דפי ה-html שלך לסיומת php
ובמקומות שבהם צריך להיות תפריט ישתמש בפקודה include, שמעתיקה את התוכן של הקובץ המאונקלד לתוך הקובץ במופעל, למשל:
page1.php

<?php include 'header.php'; ?>
  <body>
    page number 1
  </body>
<?php include 'footre.php'; ?>


page2.php
<?php include 'header.php'; ?>
  <body>
    page number 2
  </body>
<?php include 'footre.php'; ?>


header.php
<html>
  <head> <title>Yo</title> </head>


footre.php
<footer> copyright <?php echo date('Y'); ?> </footer>
</html>


התוצאה תהיה שכאשר תגלוש לעמוד page1 או לעמוד page2
לשניהם יהיה אותו תפריט ואותה תחתית, והתוכן עצמו יהיה שונה
