הוובינר עבר בהצלחה וההקלטה ניתנת לצפיה בקישור הזה.

הדוגמאות קוד מהוובינר כאן: password api, generators, iterators

מה חדש ב php5.5


נדבר על הפיצרים העיקריים
- generators + הסבר מקיף על Iterators (שרק בשבילו כבר שווה להגיע)
- password hashing api (הפונקציות החדשות לעבודה עם סיסמאות)
- getters & setters (לא נכנסו לגרסה הנוכחית)
- שאר החידושים
