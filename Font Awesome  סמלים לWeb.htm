Font Awesome - סמלים ל-Web

OrelBeY,
 01 לאפריל 2013
מגיע הזמן שבו יישומי ה-Web שלנו צריכים קצת סמלים - לנגן הווידאו שלנו, לעורך הטקסט שלנו. אנחנו צריכים משהו חינמי ומקצועי, ו-Font Awesome נותן לנו את זה.


מה זה Font Awesome ולמה צריך אותו?


הרבה פעמים אנחנו צריכים סמלים לאתרים שלנו, בין אם זה בשביל עורך הטקסט שלנו, או כדי להכניס חיים לתפריט הראשי שלנו. לצערנו, בחיפוש הראשון אנחנו מוצאים אוספי סמלים נפרדים שלא יתאימו אחד לשני, אבל אנחנו צריכים את שניהם. בחיפוש השני אנחנו מוצאים סמלים שנראים מעולים... אבל אופס - כדי להוריד אותם בגודל מעל ל-4 פיקסלים, אנחנו צריכים לשלם. אז כן, יש את האתר הנחמד dryicons ואת Open Icon Library, אבל משהו מרגיש כאן חסר... זה לא מתאים או מספיק לנו.
משמע אנחנו צריכים ספריית סמלים (בגודל נורמלי) קוד פתוח שתתאים לכל הצרכים שלנו (או לפחות לרובם). לכאן נכנס Font Awesome.
Font Awesome הוא למעשה גופן שהתווים שבו הם בעצם סמלים. (גם אותיות הם סמלים, אבל אתם מבינים למה אני מתכוון.) במקור הוא נועד לעבור בתור "תוסף" של Twitter Bootstrap (שהוא, אפשר לומר, פריימו'ורק צד לקוח לעבודה עם CSS), אבל הוא עובד מצוין גם בלעדיו.
Font Awesome מציע סמלים בשלל קטגוריות: יישום Web, עורך טקסט, סמלי כיוונים, נגן וידאו, מדיה חברתית ורפואה.

התכונות של Font Awesome


תורגם מהאתר הרשמי.

גופן אחד, 249 סמלים

באוסף אחד. Font Awesome הוא כתב ציורי של פעולות הקשורות ל-Web.

בקרת CSS

עצב בקלות את צבע הסמלים, את גודלם ועשה כל דבר אחר שאפשר עם CSS.

אין גבול לגודל של הסמלים

הסמלים הם בגרפיקה וקטורית, מה שאומר שכל סמל נראה נהדר בכל גודל שהוא.

כל זה חינם

Font Awesome הוא חינמי לחלוטין, אפילו לשימוש מסחרי.

תמיכה ב-IE 7

Font Awesome תומך אפילו ב-IE 7. אם תצטרך בכך, קבל את תנחומיי.

נועד ל-Twitter Bootstrap

Font Awesome תוכנן מההתחלה להיות מתאים לגמרי ל-Twitter Bootstrap 2.2.2.

ולבסוף... הביקורת


למרות כל השבחים שכתבתי למעלה, לא אהבתי את צורת הכתיבה של הפרויקט הזה. אני חושב שהמטרות של פריימו'ורקי CSS ושל פרויקטים כמו Font Awesome הן מעולות, אל תבינו אותי לא נכון, אבל הצורה שבה הם (כולל Font Awesome) וה-API שלהם כתובים - את זה אני ממש לא אוהב. אז כן, אני אוהב את הדברים האלה, אבל ככל הנראה אצטרך לשכתב אותם לחלוטין.
